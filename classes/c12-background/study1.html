배경 - background

배경은 컨텐츠가 없거나 높이가 없으면 배경표현이 되지 않는다.

배경스타일이 들어가는 공간은 태그의 가장 밑바닥이다.(컨텐츠박스보다 아래 깔린다)

background-color: blue; padding: 20px; height: 300px; width: 300px; box-sizing: border-box;

background-color

태그의 공간(테두리 안쪽 모든 공간)에 배경색을 나타낼 때 사용하는 css속성

background-image

배경색상 입력 - url()도구 사용

배경이미지는 입력하면 자동으로 가로세로로 반복된다.

background-repeat

배경이미지가 반복되는 속성

repeat(기본값- 가로세로로 반복)

no-repeat 반복안함

repeat-x 가로로만 반복

repeat-y 세로로만 반복

background-position

배경이미지의 위치를 변경할 때 사용하는 속성

background-position: x좌표 y좌표

키워드 방식, %방식, 픽셀방식

margin: 100px 0; height: 80vh; background-color: bisque; background-image: url(../img/kingfisher-9016787_640.jpg); background-repeat: no-repeat; /* 기본값이 가로세로 반복이기 때문에 repeat-x로 하면 가로로만 반복되고 no-repeat로 하면 반복되지 않는다 */ background-position: 100px 200px;/*기본값이 0px 0px*/
height: 100vh; background-color: aquamarine; background-image: url(../img/kingfisher-9016787_640.jpg); background-repeat: no-repeat; background-size: 300px auto; /*가로사이즈 300으로 하고 세로사이즈는 비율에 맞춘다*/ background-size: 100% auto; /*가로 전체화면크기로*/ background-size: auto 100%; background-size: cover; /*태그의 공간을 가득 채우는 방식 - 배경 이미지의 잘려나가는 부분이 생길 수 있다*/ background-size: contain; /* 이미지의 비율을 고정(이미지가 잘리지 않는 선에서) 태그의 공간에 들어갈 수 있는 최대 크기로 들어감 */ background-position: 50% 50%; /* background-position: 50% calc(50% + 10vh); 위치 가운데로 오고 세로는 위에서 10%더 내려오게한다*/ background: aquamarine url() no-repeat 0 0 /cover; /* 축약형을 많이쓴다 background-size앞에는 /슬래시 써서 구분해줘야한다 순서 외울 것 (두문자?curps)*/

cover

cover방식은 배경이미지크기를 이미지가 잘려나가도 태그의 공간을 가득 메우는 방식

contain

contain방식은 배경이미지가 잘려나가지 않는 선에서 배경이미지의 크기를 최대한 키워서 넣는 방식
